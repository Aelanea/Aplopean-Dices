## Aplopean Dice
Custom dice textures and a themed preset for the Dice So Nice! module.

## Installation

- In Foundry VTT, navigate to the Add-on Modules tab in the Setup menu.
- Click Install Module and search for "Aplopean Dice" (or paste the manifest URL: https://raw.githubusercontent.com/Aelanea/aplopean-dice/main/module.json).
- Once installed, activate the module in your world.

Requirements: This module depends on the [Dice So Nice!](https://foundryvtt.com/packages/dice-so-nice) module. Ensure it is installed and enabled.

## Usage
To apply the Aplopean dice theme:

- In your Foundry VTT world, go to Configure Settings > Module Settings > Dice So Nice! > 3D Dice Configuration.
- Under Dice Presets, select Aplopean Dices.
- Under Theme, choose Aplopean Theme.
- Click Save Settings to apply the changes.
